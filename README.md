## Installation

1. Clone the repository:

   ```bash
   git clone <repository-url>
   ```

2. Install the dependencies:

   ```bash
   npm install
   ```

## Usage

1. For development:

   ```bash
   npm run dev
   ```

2. For production:

   ```bash
   npm run build
   npm run start
   ```
